## API Conventions

All KliveIDE APIs follow consistent conventions:

- **Naming** - PascalCase for classes, camelCase for methods
- **Error Handling** - Standard exception patterns
- **Async Operations** - Promise-based asynchronous APIs
- **Events** - EventEmitter pattern for notifications
